# Smart Stock Trading Platform

A microservice platform for real-time stock trading with user authentication, order management.
Built with Golang, Docker, and monitored with Prometheus + Grafana + Loki.

## Features

- User authentication with JWT
- Place and track stock orders
- Logging with Promtail & Loki
- Metrics collection with Prometheus
- Docker ready

## Tech Stack

- **Backend:** Golang (Gin, GORM, Hexagonal Architecture)
- **Database:** Azure SQL Server
- **Containerization:** Docker, Docker Compose
- **Monitoring:** Prometheus, Grafana
- **Logging:** Loki, Promtail
